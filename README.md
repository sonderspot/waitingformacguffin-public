# WaitingForMacGuffin - Oscar Market Intelligence

Public skills and tools for [waitingformacguffin.com](https://waitingformacguffin.com) -- an Oscar prediction market intelligence platform.

## OpenClaw Skill

Install the Oscar market intelligence skill in your OpenClaw-powered agent:

```
Skills/waitingformacguffin-skill.md
```

### What it does

Two tools for real-time Oscar prediction market data:

1. **Oscar Brief** -- Quick market summary with price moves, whale trades ($1K+), frontrunner changes
2. **Oscar Research** -- Deep dive on any nominee or category with odds, precursor awards, order book depth, slippage analysis, and data-driven assessment

### Installation

Point your OpenClaw agent to this repo and install the skill from `Skills/waitingformacguffin-skill.md`.

No API keys required. All data is public and read-only.

### Example queries

- "What's happening in Oscar markets?"
- "Tell me about Chalamet"
- "What are the Best Picture odds?"
- "Should I bet on Wicked for Best Picture?"
